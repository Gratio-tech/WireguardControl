## Wireguard web interface
[![npm version](https://img.shields.io/npm/v/@gratio/wg?color=%23047dec)](https://www.npmjs.org/package/@gratio/wg)

Simple Wireguard web interface, written on TypeScript, with both ability to use already configured `.conf`-files and create a new one.

Простой web-интерфейс для VPN-wireguard с возможностью как использовать уже созданные конфиги, так и настраивать свои.

## Branches
This is the **dev** branch, here is all the development of **Wireguard-Control**.
